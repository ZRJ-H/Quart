#### RFC 10008：全新的 HTTP QUERY 方法正式发布
- **原标题**：RFC 10008: The new HTTP Query Method
- **热度**：⭐ 362 | 💬 152 评论
- **摘要**：IETF 正式发布了 RFC 10008，定义了一种新的 HTTP 方法——QUERY。与 GET（URL 传参）和 POST（body 传参但语义模糊）不同，QUERY 方法专门用于安全的、带请求体的查询操作，解决了 RESTful API 设计中「查询语义」与 HTTP 动词不匹配的长期痛点。这是 HTTP 协议的一次重要演进。
- **链接**：[RFC Editor](https://www.rfc-editor.org/info/rfc10008/)｜[HN 讨论](https://news.ycombinator.com/item?id=48568502)

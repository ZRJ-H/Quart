## 今日精选

### 1. 全模态主动感知代理：将视频理解建模为推理过程
- **原标题**: Native Active Perception as Reasoning for Omni-Modal Understanding
- **作者/机构**: Zhenghao Xing, Ruiyang Xu, Yuxuan Wang 等（CUHK / 阿里巴巴通义实验室）| [arXiv](http://arxiv.org/abs/2606.19341)
- **核心贡献**: ① 提出 OmniAgent，首个原生全模态代理，将视频理解形式化为 POMDP 驱动的「观察-思考-行动」迭代循环，按需选取视听线索并写入持久文本记忆，使推理复杂度与视频时长解耦。② 训练方法：Agentic SFT（best-of-N 轨迹合成 + 双阶段质量控制）+ Agentic RL（TAURA 算法，用回合级熵值评估关键发现回合）。③ 在 LVBench 上，7B 模型（50.5%）超越 10 倍参数量的 Qwen2.5-VL-72B（47.3%），展现出正向推理时扩展趋势。
- **工程关联**: 对 Agent 开发（多模态感知 + 推理循环设计）、模型部署（7B 模型替代 72B 模型降本）有直接参考价值。
- **价值点**: 通过主动感知降低多模态推理成本，7B 模型即可在视频理解上超越 72B 闭源模型，开源代码可直接复用。已被 ICML 2026 接收。

### 2. 重新思考奖励监督：基于评分标准的自蒸馏方法
- **原标题**: Rethinking Reward Supervision: Rubric-Conditioned Self-Distillation
- **作者/机构**: Siyi Gu, Jialin Chen, Sophia Zhou 等（Yale / NYU）| [arXiv](http://arxiv.org/abs/2606.19327)
- **核心贡献**: ① 指出现有推理模型后训练方法的局限：知识蒸馏依赖昂贵的 CoT 标注，标量奖励信号（如 GRPO）则忽略细粒度的过程改进。② 提出「评分标准条件的自蒸馏」，用教师模型基于准则级评分标准（rubric）对学生模型自身采样的轨迹进行 token 级指导，实现更精细的信用分配。③ 在多个科学推理基准上平均超越 GRPO 1.0 分、超越 OPSD 0.9 分。
- **工程关联**: 直接改善 LLM 后训练流程，可替代 GRPO 等 RL 方法，实现更高效的推理能力注入。
- **价值点**: 提供了一种比标量奖励更可解释、更精细的推理模型训练范式，适合在生产环境中对模型进行定向能力增强。

### 3. 扩散证明：超越自回归生成的定理证明新范式
- **原标题**: Diffusion-Proof: Recipe for Formal Theorem Proving Beyond Auto-Regressive Generation
- **作者/机构**: Ruida Wang, Rui Pan, Pengcheng Wang 等（清华 / 上海 AI Lab）| [arXiv](http://arxiv.org/abs/2606.19315)
- **核心贡献**: ① 首次将扩散大语言模型（dLLM）应用于形式化定理证明，提出两个 7B 模型：dLLM-Prover-7B 负责长程连贯的全证明生成，dLLM-Corrector-7B 利用扩散填空能力进行双向局部证明修正。② 在 ProofNet 和 MiniF2F 上分别相对自回归基线提升 1.61% 和 6.14%（绝对提升）。③ 成功解决一道 DeepSeek-Prover-V2-7B 未能解决的 IMO 题目，展示了扩散模型在保持长程数学推理连贯性上的独特优势。
- **工程关联**: 拓展了推理模型的技术路线（扩散生成 vs 自回归），对数学推理、代码生成等需要长程依赖的场景有参考意义。
- **价值点**: 证明扩散模型在需要全局一致性的长上下文推理任务中具有不可替代的优势，为 Agent 推理和自动证明提供了新的技术路线。

### 4. 用程序合成解释注意力头
- **原标题**: Explaining Attention with Program Synthesis
- **作者/机构**: Amiri Hayes, Belinda Li, Jacob Andreas（MIT）| [arXiv](http://arxiv.org/abs/2606.19317)
- **核心贡献**: ① 提出将 Transformer 的注意力头近似为可执行的 Python 程序，管线为：计算注意力矩阵 → 用预训练 LM 生成候选程序 → 基于 held-out 预测保真度重排序。② 仅需不到 1000 个程序即可复现 GPT-2、TinyLlama-1.1B、Llama-3B 中注意力头的模式，平均 IoU 相似度超 75%。③ 替换 25% 的注意力头为程序化代理，仅带来 16% 的平均困惑度增加，下游 QA 性能基本保持。
- **工程关联**: 对 LLM 可解释性、模型压缩和推理机制的符号化分析有重要价值。
- **价值点**: 提供了一种将黑箱注意力机制转化为人类可读程序的可扩展管线，有助于模型调试、轻量化部署和安全审计。

### 5. 数据智能代理：基于自主编码代理的企业数据系统
- **原标题**: Data Intelligence Agents: Interpreting, Modeling, and Querying Enterprise Data via Autonomous Coding Agents
- **作者/机构**: Anoushka Vyas, Aarushi Dhanuka, Sina Khoshfetrat Pakazad, Henrik Ohlsson | [arXiv](http://arxiv.org/abs/2606.19319)
- **核心贡献**: ① 提出 DIA——三个代理（数据解释器、模式创建器、查询生成器）组成的系统，将自主编码代理（ACA）作为一等抽象：代理生成、执行、验证并修复具体产物，通过共享记忆复用经验。② 已在企业生产环境部署。③ 在 7 个 SQL 基准（覆盖 4 类任务、4 种方言）的全自主模式下，匹敌或超越所有已知最佳结果。
- **工程关联**: 对多 Agent 系统设计（共享记忆、代码生成-执行-验证循环）、Text-to-SQL 和企业级 AI 应用部署有直接参考价值。
- **价值点**: 证明了基于执行闭环和共享记忆的多 Agent 架构可以在复杂企业数据环境中达到生产级水平，是 Agent 工程落地的优秀范例。

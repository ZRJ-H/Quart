## 今日精选

### 1. STARE：基于惊讶度的Token级优势重加权实现策略熵稳定训练

**原标题**：STARE: Surprisal-Guided Token-Level Advantage Reweighting for Policy Entropy Stability
**作者/机构**：Haipeng Luo 等（清华深研院 / 上海AI Lab）
**链接**：[arXiv](http://arxiv.org/abs/2606.19236)

**核心贡献**：
针对 GRPO 在 LLM 推理训练中常见的「策略熵崩塌」问题，作者首次从一阶梯度分析揭示了 token 级信用分配错配机制——每 token 的熵变化可分解为轨迹级优势与熵敏感度函数的乘积，形成「优势-惊讶度」四象限结构。基于此提出 STARE 方法：通过批次内惊讶度分位数筛选熵关键 token 子集，选择性重加权其有效优势，并引入目标熵闭环门控稳定熵调控。在 1.5B 到 32B 多尺度模型、三类任务（Short CoT / Long CoT / 多轮工具调用）上验证，STARE 可在数千步训练中维持熵在目标区间内，AIME24/AIME25 上平均准确率比 DAPO 等基线高 4%-8%。代码已开源：github.com/hp-luo/STARE。

**与AI Engineering的关联**：
对正在或计划使用 GRPO 进行推理模型后训练的团队有直接参考价值——STARE 提供了一个即插即用的熵稳定方案，可避免训练中途策略退化，尤其适用于长推理链和多轮工具调用场景。

### 2. 重新思考奖励监督：基于评分标准的条件自蒸馏

**原标题**：Rethinking Reward Supervision: Rubric-Conditioned Self-Distillation
**作者/机构**：Siyi Gu 等（耶鲁大学）
**链接**：[arXiv](http://arxiv.org/abs/2606.19327)

**核心贡献**：
现有推理模型后训练中，蒸馏依赖昂贵且可能含有噪声的思维链标注，而 RL 将评估反馈压缩为标量信号丢失细节。本文提出「评分标准条件自蒸馏」框架：用细粒度评分标准（rubric）替代单一参考解，教师模型基于评分标准对学生的采样轨迹提供 token 级指导。两阶段流程——先学习生成任务专属评分标准，再训练评分标准引导的推理器——在多项科学推理基准上平均超过 GRPO 1.0 分、超过 OPSD 0.9 分。

**与AI Engineering的关联**：
提供了一种将结构化评估准则转化为细粒度训练信号的方法论，优于纯 RL 标量奖励，适用于需要精确控制推理质量的后训练流水线。

### 3. 用程序合成解释注意力机制

**原标题**：Explaining Attention with Program Synthesis
**作者/机构**：Amiri Hayes 等（MIT CSAIL，Jacob Andreas 组）
**链接**：[arXiv](http://arxiv.org/abs/2606.19317)

**核心贡献**：
提出一种将 Transformer 注意力头行为近似为可执行 Python 程序的可扩展管道：对每个注意力头，在随机样本上计算注意力矩阵，用 LLM 生成可复现注意力模式的候选程序，再在 held-out 数据上排序筛选。不到 1000 个生成程序即可复现 GPT-2、TinyLlama-1.1B 和 Llama-3B 的注意力模式，在 TinyStories 上平均 IoU 超过 75%。将 25% 注意力头替换为程序化代理后，困惑度仅上升 16%，下游 QA 任务性能基本保持。

**与AI Engineering的关联**：
直接将 LLM 内部组件解释为可读代码，对模型调试、安全检查、知识编辑和轻量化部署有重要参考价值。反向工程注意力头的方法论可能推广到 MLP 层等其他组件。

### 4. Diffusion-Proof：超越自回归生成的扩散形式化定理证明

**原标题**：Diffusion-Proof: Recipe for Formal Theorem Proving Beyond Auto-Regressive Generation
**作者/机构**：Ruida Wang 等（香港科技大学）
**链接**：[arXiv](http://arxiv.org/abs/2606.19315)

**核心贡献**：
首次将扩散语言模型（dLLM）应用于形式化定理证明领域，解决自回归模型长程序列误差累积和全局一致性问题。框架包含两个 7B 模型：dLLM-Prover-7B 完成整体证明书写，dLLM-Corrector-7B 利用扩散模型的填空能力进行局部双向修正。在 ProofNet-Test 和 MiniF2F-Test 上分别提升 1.61% 和 6.14%（绝对值），并成功解决了一道 DeepSeek-Prover-V2-7B 未能攻克的 IMO 题目。

**与AI Engineering的关联**：
探索了扩散模型在需要长程一致性的复杂推理任务中的潜力，为 LLM 架构选型提供了超越自回归的新思路。对形式化验证、代码生成等需要全局结构优化的场景具有启示意义。

### 5. 什么使潜在思维链监督有效：信息论视角分析

**原标题**：What Makes Effective Supervision in Latent Chain-of-Thought: An Information-Theoretic Analysis
**作者/机构**：Xinghao Chen 等（香港理工大学 / 华为诺亚方舟实验室）
**链接**：[arXiv](http://arxiv.org/abs/2606.20075)

**核心贡献**：
潜在思维链（Latent CoT）将推理过程内化于连续隐状态，但结果监督信号弱、隐轨迹易语义漂移。本文从信息论角度将此失效诊断为「双重坍缩」——优化路径梯度衰减 + 隐空间表示漂移，并将过程监督分解为轨迹监督和空间监督两个互补维度。提出统一隐探针（ULP）度量隐轨迹与显式推理步骤间的互信息，实验揭示「信息-性能绑定」规律：推理准确率取决于隐链中的信息保真度。代码已开源：github.com/EIT-NLP/Supervision-in-Latent-CoT。

**与AI Engineering的关联**：
为高效推理（不依赖冗长显式思维链）的训练方法提供了理论框架。对于追求推理速度与质量平衡的部署场景——如边缘设备、流式推理——Latent CoT 是重要方向，本文给出了可操作的监督设计原则。

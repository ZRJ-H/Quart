## 今日精选

### 1. UltraQuant：面向上下文密集型 Agent 的 4-bit KV 缓存压缩
- **原标题**: UltraQuant: 4-bit KV Caching for Context-Heavy Agents
- **作者/机构**: Inesh Chakrabarti 等（AMD）| [arXiv](https://arxiv.org/abs/2606.20474)
- **核心贡献**: ① 解决 Agent 长上下文多轮对话场景下 KV 缓存内存压力大的问题；② 提出 4-bit KV 缓存压缩方案，采用 Walsh-Hadamard 旋转、非对称 K/V 处理、FP4 近似路径及 AMD CDNA4 原生 scaled-MFMA 支持；③ 在长上下文多轮 Agent 负载上，P50 首 token 延迟降低 3.47 倍（缓存压力大的后期轮次），整体吞吐提升 1.63 倍（对比 FP8 KV 基线）
- **工程关联**: 直接面向 Agent 推理部署优化，对生产环境中运行长上下文 Agent 系统的服务成本与延迟有重大参考价值
- **价值点**: 为 Agent 服务基础设施提供可落地的 4-bit KV 缓存方案，在保持任务质量的同时大幅降低延迟和内存占用

### 2. 探测-精炼调优：为编程 Agent 迭代优化仓库指导文件
- **原标题**: Probe-and-Refine Tuning of Repository Guidance for Coding Agents
- **作者/机构**: Asa Shepard, Jeannie Albrecht（Williams College）| [arXiv](https://arxiv.org/abs/2606.20512)
- **核心贡献**: ① 解决 AGENTS.md 等仓库指导文件对编程 Agent 是否真正有效这一争议；② 提出「探测-精炼调优」方法——通过合成 bug 修复探测任务，用单次 LLM 调用迭代诊断并修补指导文件，无需 Agent 循环或工具调用；③ SWE-bench Verified 上 Qwen3.5-35B-A3B 的 resolve rate 从基线的 25.5% 提升至 33.0%（p<0.001），且改进来自覆盖率（+14.5pp 可评估补丁）而非精度
- **工程关联**: 直接提升编程 Agent（如 SWE-bench、Cursor、Copilot 等场景）在大型代码仓库中的定位与修复能力
- **价值点**: 提供了一种零额外成本的 AGENTS.md 自动优化方法，任何使用编程 Agent 的团队都可以立即应用

### 3. DiffusionGemma 有多透明？扩散语言模型的可解释性深度研究
- **原标题**: How Transparent is DiffusionGemma?
- **作者/机构**: Joshua Engels, Neel Nanda, Rohin Shah 等（Google DeepMind）| [arXiv](https://arxiv.org/abs/2606.20560)
- **核心贡献**: ① 系统研究扩散语言模型 DiffusionGemma 的推理透明度，将其分解为「变量透明度」（中间计算状态是否可理解）和「算法透明度」（是否可重建推理过程）；② 发现通过可解释 token 瓶颈映射，可将不透明串行深度从 Gemma 4 的 28.6 倍降至仅 1.1 倍，且不损失性能；③ 首次揭示扩散模型特有的推理现象：非时序推理（non-chronological reasoning）、token/序列涂抹（smearing）、中间上下文推理
- **工程关联**: 对 LLM 可解释性、安全对齐、模型调试方向有深远影响——理解扩散模型如何「思考」是安全部署的前提
- **价值点**: 为扩散语言模型的可解释性建立了评估框架和基准，对理解和审计新型 LLM 架构至关重要

### 4. 你的鼠标和视线偷偷泄露了偏好：利用隐式反馈做 LLM 对齐
- **原标题**: Your Mouse and Eyes Secretly Leak Your Preference: LLM Alignment using Implicit Feedback from Users
- **作者/机构**: Haw-Shiuan Chang 等（UMass Amherst）| [arXiv](https://arxiv.org/abs/2606.20482)
- **核心贡献**: ① 构建 IFLLM 数据集，从 59 名众包用户的多轮对话中采集鼠标轨迹和眼球注视数据作为隐式偏好信号；② 基于隐式反馈的奖励模型将纯文本奖励模型的准确率从 55% 提升至 64%；③ DPO 对齐后，响应质量提升幅度接近纯文本方法的三倍，证明了隐式信号在真实场景中的巨大价值
- **工程关联**: 为 RLHF/DPO 对齐提供了低成本、可规模化的人类反馈采集新范式，不再依赖昂贵的显式标注
- **价值点**: 开源数据集和代码，提出了一种利用用户自然行为信号（鼠标+视线）替代显式反馈的对齐方案，可大幅降低对齐成本

### 5. 边际优势累积：面向记忆驱动 Agent 自我进化的跨批次证据聚合
- **原标题**: Marginal Advantage Accumulation for Memory-Driven Agent Self-Evolution
- **作者/机构**: Mingyu Yang 等 | [arXiv](https://arxiv.org/abs/2606.20475)
- **核心贡献**: ① 解决批量轨迹蒸馏中同一记忆操作可能在不同批次收到矛盾反馈的问题；② 提出 MAA（边际优势累积），通过构建跨批次可比较的差分信号、EMA 累积签名证据、语义身份合并实现跨批次可追溯性；③ 在 4 个基准、4 个目标模型的 16 组设置中，14 组达到最优，同时将优化阶段的 token 消耗降低约 75%
- **工程关联**: 为 Agent 记忆系统的自我进化提供了稳健的证据累积机制，适用于需要长期运行和持续学习的 Agent 系统
- **价值点**: 可作为后处理架构接入现有 Agent 记忆蒸馏流程，以 75% 更少的 token 消耗实现更优的记忆进化效果

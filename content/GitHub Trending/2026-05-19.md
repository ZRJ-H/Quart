## 🔍 深度分析

### 🏆 今日动量之王：tinyhumansai/openhuman
+3,941⭐（13.7K → 17.6K），增长率 28.8%。

openhuman 的爆发有三个核心驱动力：

1. **桌面端 AI 助手赛道成熟** — OpenClaw（372.7K⭐）已经验证了"个人 AI 助手"的需求，openhuman 作为更易用、更原生的替代品（Rust + 原生桌面端 vs OpenClaw 的 TypeScript/Electron），吸引了大量 OpenClaw 用户的注意。

2. **零训练期设计** — 与 hermes-agent（需要学习循环来"成长"）不同，openhuman 开箱即用，直接感知工作上下文。对于不想"养 Agent"的用户来说，这是关键差异化。

3. **Karpathy 工作流背书** — openhuman 的工程设计灵感来自 Andrej Karpathy 的 LLM Knowledgebase 工作流，这位特斯拉前 AI 总监、OpenAI 创始成员的背书在开发者社区有极强号召力。

openhuman 总星 17.6K 相对于它的日增 +3,941 来说仍然很小，预计未来 3-5 天将继续保持高增长。

### 🗺️ 值得关注的赛道

1. **Claude Code 生态全面爆发（第三周）** — ECC（+1,960⭐）、opencode（+1,050⭐）、hermes-agent（+3,481⭐）、agent-skills（+1,244⭐）四箭齐发。Claude Code 用户基数扩张速度已远超其他 Agent 平台，周边工具链进入红利收割期第三周。关键变化：从"基础设施"（opencode、ECC）转向"内容/技能生态"（agent-skills、academic-research-skills）。

2. **边缘 AI 工具独立赛道形成** — supertonic（+1,776⭐，TTS）、openhuman（Rust 本地推理）代表"小模型+高性能+零云端依赖"新范式。2026 年下半年，预计会有更多"设备端 AI Agent"项目爆发。

3. **开源情报（OSINT）工具崛起** — Shadowbroker（+767⭐）和 RuView（WiFi 雷达感知，+700⭐）代表"个人化全球态势感知"需求。这类项目以往是小众黑客工具，现在进入 GitHub Trending，说明地缘政治紧张正在推动开发者社区关注"自己动手做情报"。

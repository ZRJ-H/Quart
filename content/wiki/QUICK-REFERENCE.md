## 更新流程（自动执行）

```
┌─────────────────────────────────────────────────────────────┐
│  每日Wiki更新流程                                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  0. 同步GitHub项目档案（新增）                               │
│     ├── 检查/GitHub 项目档案/目录                            │
│     ├── 复制新文件到wiki/entities/                           │
│     └── 添加type字段，更新last_seen                          │
│                                                             │
│  1. 读取今日数据                                             │
│     ├── 时政要闻/{日期}.md                                  │
│     ├── AI科技动态/{日期}.md                                │
│     └── GitHub Trending/{日期}.md                           │
│                                                             │
│  2. 创建来源摘要页                                           │
│     └── wiki/sources/{日期}-{来源名}.md                     │
│                                                             │
│  3. 识别新实体/概念                                          │
│     ├── 新项目/产品 → 创建实体页                            │
│     ├── 新政策/趋势 → 创建概念页                            │
│     └── 已有实体 → 更新事件时间线                           │
│                                                             │
│  4. 创建/更新实体和概念页                                    │
│                                                             │
│  5. 更新索引和日志                                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

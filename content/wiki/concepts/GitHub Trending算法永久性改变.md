# GitHub Trending算法永久性改变

## 定义
GitHub Trending页面算法可能已发生永久性改变的判断。核心证据：经典异常项目（freeCodeCamp 448K⭐、puppeteer 94.9K⭐）连续3天稳定在榜，而非此前偶发性的单日异常。freeCodeCamp 3日增长轨迹（+146→+736→+633）和puppeteer首次异常入榜（+56⭐）共同暗示：GitHub Trending的权重算法可能已调整，经典大项目的「基础曝光权重」被系统性提升。同时榜单总规模缩减（周三仅13个项目，低于正常工作日18-25个）。

## 核心要点
- freeCodeCamp连续第3天在榜（448.6K⭐，日增稳定在600+）
- puppeteer（94.9K⭐）首次以异常模式入榜
- 经典异常从「偶发」变为「持续3天」
- 周三仅13个项目，低于正常工作日水平
- 可能永久性改变GitHub项目曝光生态

## 相关政策/事件
- freeCodeCamp连续3天稳定上榜（6/15-6/17）
- puppeteer 6/17首次异常入榜

## 实际应用
- 开源项目的GitHub曝光策略需重新评估
- 小项目的「隐藏宝石」发现价值相对提升
- 需持续跟踪算法调整后的长期影响

# Vibe Coding

## 定义
Karpathy 在 2025 年创造的术语，指一种完全依赖 AI Agent 生成代码的编程方式——开发者描述想要什么，由 Agent 编写代码、修复错误、完成部署。Vibe Coding 的核心是"提升编程的下限"，让几乎任何人都能创建软件。

## 核心要点
- 由 Karpathy 在 2025 年创造并迅速流行
- 与 [[Agentic-Engineering]] 互补：Vibe Coding 提升下限，Agentic Engineering 提升上限
- 适合原型和个人工具，但专业团队需要 Agentic Engineering
- 不是"降低质量"，而是让非程序员也能参与软件开发
- 在 Software 3.0 范式下，Vibe Coding 是最直接的入门方式

## 相关政策/事件
- 2025 年 Karpathy 首次使用该术语，迅速成为 AI 编程领域的流行词
- 2026 年 Sequoia Ascent 上 Karpathy 明确区分 Vibe Coding 和 Agentic Engineering
- 2026-06-19：智谱GLM-5口号"From Vibe Coding to Agentic Engineering"将这一演进作为产品定位，GitHub Trending #9 +286⭐

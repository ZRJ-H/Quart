# AI编程工具军备竞赛

## 定义
2026年6月AI编程工具赛道进入'军备竞赛'阶段的标志：Anthropic Fable 5发布仅48小时后，小米以MIT许可证开源MiMo Code，SWE-Bench Pro V2领先Claude Code 5pp，并支持零成本迁移Claude Code配置。AI编程工具竞争节奏从'年'加速到'天'。

## 核心要点
- Fable 5 6/9发布→MiMo Code 6/11开源（仅间隔48小时）
- MIT开源 vs Claude Code闭源的路线之争
- Claude Code零成本迁移策略降低用户切换门槛
- 中国公司首次进入终端AI编程工具竞争
- SWE-Bench性能优势可能来自工程优化而非模型本身
- 竞争焦点：记忆系统/Compose模式/语音交互/本地化

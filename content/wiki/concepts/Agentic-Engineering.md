# Agentic Engineering

## 定义
Karpathy 提出的专业工程技术学科，指在保持软件质量（正确性、安全性、可维护性）的前提下，协调不可靠但强大的 AI Agent 来完成复杂工作的系统方法论。与 Vibe Coding（提升编程下限）相对，Agentic Engineering 专注于提升专业开发者的上限。

## 核心要点
- 与 Vibe Coding 互补：Vibe Coding 让任何人都能做软件，Agentic Engineering 让专家做得更好更快
- 核心技能不是记住 API 细节，而是理解底层概念（存储、视图、不可变性、安全边界、系统架构）
- Agentic Engineer 的职责：设计规范、监督计划、审查 diff、编写测试、创建评估循环、管理权限、隔离工作区、维护质量
- "10x 工程师"概念可能被大幅放大——掌握 Agentic 工作流的人可能远超 10x
- 面试应测试 Agentic Engineering 能力：用 Agent 构建完整项目、部署、安全加固、防御对抗性 Agent

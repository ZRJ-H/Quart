# IT之家-2026-06-12-华为HDC开幕

## 摘要
华为HDC 2026于6月12日开幕，HarmonyOS 7正式发布。系统内核嵌入盘古大模型6.0，首发Harmony Agent Framework 2.0。华为同时宣布开源盘古大模型2.0（505B MoE/512K上下文）。

## 关键要点
- HarmonyOS 7成首个完成AI化改造的操作系统
- Harmony Agent Framework 2.0意图识别成功率超90%
- 小艺日活1.8亿，日均唤醒30亿次
- 星盾防诈平台全面升级
- 盘古大模型2.0开源（Pro版505B/Flash版92B）
- Mate90系列秋季首发

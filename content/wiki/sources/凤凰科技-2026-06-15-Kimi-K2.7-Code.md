# 凤凰科技：Kimi K2.7 Code开源，1.1T参数编程模型

## 摘要
月之暗面6/12发布并开源Kimi K2.7 Code编程模型（1.1万亿参数MoE，256K上下文）。核心升级：长上下文编程指令遵循提升、token消耗减30%、Code Bench v2性能提升21.8%。6倍速高速版今日开放API。

## 关键要点

- 1.1T参数MoE架构，MIT协议开源
- token消耗减少30%，缓解过度思考问题
- Code Bench v2性能提升21.8%
- 6倍速高速版6/15开放API

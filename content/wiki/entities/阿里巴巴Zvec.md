# Zvec（阿里巴巴嵌入式向量数据库）

## 基本信息
- **类型**: 开源项目/向量数据库
- **开发者/主体**: 阿里巴巴 (alibaba)
- **GitHub**: https://github.com/alibaba/zvec
- **当前版本**: v0.5.0（2026-06-12）

## 核心特性
- 轻量级嵌入式向量数据库
- SQLite式架构：pip install zvec即可使用
- 底层基于阿里生产验证的Proxima向量引擎
- v0.5.0新增：原生全文检索、混合查询、DiskANN算法
- Go/Rust SDK
- 支持手机端(Dart/Flutter)、浏览器(WASM)及MCP集成
- 可视化工具Zvec Studio
- 号称"向量搜索界的SQLite"

# AWS Lambda MicroVMs

## 基本信息
- **类型**: Serverless计算功能
- **开发者/主体**: AWS (Amazon Web Services)
- **主要功能**: 在Lambda中运行完全隔离的沙箱环境，拥有完整生命周期控制权

## 核心特性
- Lambda首次引入MicroVM隔离机制
- 完全隔离的沙箱环境
- 用户拥有完整生命周期控制权
- 保持Lambda轻量特性同时提升隔离性和安全性
- Serverless基础设施的重大演进

# AI科技动态日报 — 2026年6月26日（星期五）

## 今日焦点

本周（6月24-26日）全球AI产业多线并进：OpenAI与博通联合发布首款自研AI推理芯片Jalapeño，Anthropic IPO进程持续推进，Sakana AI以Fugu Ultra多智能体编排系统另辟蹊径在多项基准上超越GPT-5.5/Opus 4.8。中国方面，阿里发布首个原生语言世界模型Qwen-AgentWorld，交通运输部等五部门印发AI+交通行动方案，智谱科创板IPO进入冲刺阶段。AI芯片竞赛、多智能体编排、AI安全与出口管制交织成为本周主旋律。

## 重大事件

### OpenAI联合博通发布首款自研AI推理芯片Jalapeño
- **来源**: OpenAI官方 / Bloomberg / CNN | 链接: https://openai.com/index/openai-broadcom-jalapeno-inference-chip/
- **摘要**: 6月24日，OpenAI正式推出与博通联合设计的首款定制AI推理芯片Jalapeño。该芯片专为ChatGPT和Codex等推理负载优化，9个月完成流片，成本较采购GPU方案节省约50%。OpenAI与博通此前宣布合作建设10GW算力基础设施，Jalapeño为合作首个成果，计划于2026年底部署。
- **价值点**: 标志着OpenAI从模型公司向AI基础设施全栈玩家转型的关键一步，降低对英伟达GPU的依赖，可能重塑AI推理芯片市场格局。

### Sakana AI发布Fugu Ultra多智能体编排系统，多项基准超GPT-5.5与Opus 4.8
- **来源**: Sakana AI官方 / 腾讯新闻 | 链接: https://sakana.ai/fugu
- **摘要**: 6月22日，日本东京AI初创Sakana AI推出Fugu（河豚）多智能体编排系统及旗舰版Fugu Ultra。该系统不训练新基座模型，而是通过70亿参数强化学习协调器动态调度多个专家模型，以单一API提供服务。在SWE-Bench Pro中Fugu Ultra得分73.7%，远超GPT-5.5（58.6%）和Opus 4.8（69.2%），并在GPQA-D、TerminalBench等多项测试中登顶。在禁止使用Fable 5/Mythos 5的约束下仍取得该成绩。定价仅为Opus 4.8的三分之一到二分之一。
- **价值点**: 验证了"多模型调度＞更大单体模型"的路线，在AI出口管制加剧背景下，可通过编排开源模型获得接近前沿闭源模型的性能，具有地缘战略意义。

### Anthropic秘密提交S-1启动IPO，估值9650亿美元
- **来源**: CNBC / Yahoo Finance / Anthropic官方 | 链接: https://anthropic.com/news/confidential-s-1
- **摘要**: Anthropic于6月1日向SEC秘密提交S-1草案，正式启动IPO进程，目标估值1.75-1.8万亿美元，计划募资高达750亿美元，有望成为史上最大IPO之一（仅次于SpaceX和沙特阿美）。此前Anthropic在5月完成650亿美元H轮融资，投后估值9650亿美元，超越OpenAI成为全球估值最高AI初创。公司年化收入已超470亿美元，预计6月底达500亿美元。
- **价值点**: Anthropic的IPO将成为AI行业资本化进程的标志性事件，与SpaceX、OpenAI共同构成2026年"万亿美元IPO潮"。

### Anthropic应美国政府要求暂停Fable 5与Mythos 5访问
- **来源**: RFI / iThome / Anthropic官方声明 | 链接: https://www.rfi.fr/cn/%E5%9B%BD%E9%99%85/20260613
- **摘要**: Anthropic于6月12日应美国商务部要求，紧急暂停新发布的Fable 5和Mythos 5两款顶级模型的访问权限。美国政府援引出口管制条例，要求切断任何外籍用户（包括Anthropic外籍员工）访问。Anthropic不同意该决定但被迫合规，称"仅因发现潜在规避手段就召回已部署商业模型不合理"，正努力尽快恢复。Opus 4.8及其他模型不受影响。
- **价值点**: 这是美国政府首次对已部署的AI模型实施出口管制，为全球AI治理树立争议性先例，直接催生Sakana Fugu等替代方案需求。

### 阿里千问发布首个原生语言世界模型Qwen-AgentWorld
- **来源**: 阿里千问官方 / IT之家 | 链接: https://qwen.ai/blog?id=qwen-agentworld
- **摘要**: 6月24日，阿里巴巴通义千问发布Qwen-AgentWorld，业界首个原生语言世界模型（LWM）。该模型从继续预训练阶段就将环境建模作为训练目标，覆盖MCP、Search、Terminal、SWE、Web、OS、Android七大交互领域，基于超1000万条真实轨迹训练，提供35B-A3B与397B-A17B两规格。在AgentWorldBench评测中，397B版本整体得分58.71，超越GPT-5.4（58.25）。模型已在Hugging Face和ModelScope开源。
- **价值点**: 开辟了"先预测、再行动"的通用智能体新范式，语言世界模型可能成为下一阶段Agent竞赛的关键变量。

### OpenAI更新GPT-5.5 Instant，宣布停用o3与GPT-4.5
- **来源**: OpenAI Help Center / 官方博客 | 链接: https://help.openai.com/zh-hant-hk/articles/9624314
- **摘要**: 6月24日，OpenAI更新ChatGPT默认模型GPT-5.5 Instant，优化意图理解、复杂约束处理和本地推荐能力，回应更简洁自然（字数减少30.2%）。同时宣布GPT-4.5将于6月27日从ChatGPT下线，o3将于8月26日下线，仅影响ChatGPT，不影响API。
- **价值点**: 反映OpenAI加速模型迭代换代、淘汰旧模型的策略，GPT-4.5退役标志着o系列到5系列的代际迁移加速。

### Google推出Gemini 3.5 Flash内置Computer Use功能
- **来源**: Google Blog | 链接: https://blog.google/innovation-and-ai/models-and-research/gemini-models/introducing-computer-use-gemini-3-5-flash/
- **摘要**: 6月24日，Google为Gemini 3.5 Flash正式推出Computer Use能力，允许模型直接操控计算机界面完成多步骤任务。Gemini 3.5 Flash发布于5月Google I/O 2026，在Agentic和Coding任务中超越上一代Gemini 3.1 Pro，输出速度快4倍。

### 五部门联合印发《"人工智能+交通运输"典型应用场景创新行动方案》
- **来源**: 交通运输部 / 经济参考报 | 链接: https://www.mot.gov.cn/xinwen/jiaotongyaowen/202606/t20260625_4208193.html
- **摘要**: 6月25日，交通运输部等五部门发布行动方案，聚焦智能驾驶、智慧公路、智能铁路、智慧民航、智能安全监管等十大方向，到2030年实现AI在交通领域规模化创新应用。方案提出开展"端到端"大模型研发测试、公路自由流收费、船舶编队智能航行等41个重点场景，将组织实施超百项试点项目。
- **价值点**: 标志着中国"AI+交通"从单点试点迈入系统性规模化新阶段，为智能驾驶和交通大模型打开政策空间。

### 智谱科创板IPO推进：6月22日股东大会审议发行方案
- **来源**: 21经济网 / 证券时报 / 观察者网 | 链接: https://www.21jingji.com/article/20260602/herald/1b90a42b49d0bccbf55c9af1f3347288.html
- **摘要**: 智谱（02513.HK）科创板上市进入关键阶段，6月22日年度股东大会审议A股发行方案。拟发行不超过3876.9万股新A股（占发行后总股本2%-8%），募资不超过150亿元，其中120亿元投入AI通用基座大模型。智谱1月8日港股上市募资约49亿港元，至5月底仅动用约20.58亿港元。MiniMax紧随其后启动A股辅导。"AI六小虎"分化加速，百川智能和零一万物转向垂类市场。
- **价值点**: "A+H"双融资平台模式成型，内地资本市场深度参与大模型产业链定价，AI产业与资本市场形成正向循环。

### 八部门联合发布《关于加快"人工智能+消费"发展的实施意见》
- **来源**: 中国工业新闻网 | 链接: https://www.cinn.cn/yc/2026/06-22/V1QP36Wk.html
- **摘要**: 6月18日，商务部等8部门发布17条举措推动AI+消费，从AI手机、智能家居到人形机器人、脑机接口全面覆盖。预计2026年中国消费级AI硬件（不含手机和汽车）市场规模将突破1.27万亿元，人形机器人产量将达10-20万台级别。
- **价值点**: AI+消费上升为国家战略行动，短期看智能家居和可穿戴设备率先放量，中长期人形机器人有望复刻新能源汽车培育路径。

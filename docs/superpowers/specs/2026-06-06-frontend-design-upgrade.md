# 前端设计升级规范

## 概述

将 Quartz 网站从默认样式升级为 Editorial/Magazine（杂志风）设计风格，采用"优雅渐进"方案，在不破坏现有功能的前提下显著提升视觉品质。

## 命名规范

- 网站标题：DogeLover 知识库（无 emoji 前缀）

## 设计方向

**风格**：Editorial/Magazine（杂志风）
**理念**：优雅、精致、有呼吸感
**参考**：frontend-design skill 的设计原则——避免 "AI slop"，追求独特性和品质感

## 设计规范

### 配色方案

保留蓝灰色系的专业感，增加金色点缀提升精致度。

| 角色 | 颜色 | 用途 |
|------|------|------|
| Primary | #284b63 | 主色调、标题、链接 |
| Primary Light | #3a6a8a | 悬停状态 |
| Accent | #c9a84c | 金色点缀、高亮、标签 |
| Accent Light | #d4b86a | 金色悬停 |
| Background | #faf8f8 | 页面背景 |
| Text | #2b2b2b | 正文颜色 |
| Text Secondary | #4e4e4e | 次要文字 |
| Border | #e5e5e5 | 边框、分隔线 |

### 排版系统

**字体组合**：
- 标题：Noto Serif SC（思源宋体）— 优雅而有力量
- 正文：Inter — 清晰易读，现代感强
- 代码：SF Mono / Fira Code

**排版参数**：
- 标题字间距：0.02em
- 标题行高：1.3
- 正文行高：1.7
- 段落间距：1rem

### 阴影系统

| 级别 | 值 | 用途 |
|------|-----|------|
| sm | 0 1px 2px rgba(0,0,0,0.05) | 卡片默认 |
| md | 0 4px 6px rgba(0,0,0,0.07) | 悬停状态 |
| lg | 0 10px 15px rgba(0,0,0,0.1) | 弹出层 |

### 过渡系统

| 级别 | 值 | 用途 |
|------|-----|------|
| fast | 150ms ease | 按钮、链接 |
| normal | 250ms ease | 卡片、边框 |
| slow | 400ms ease | 页面动画 |

### 间距系统

使用 4px 基准的间距系统：
- xs: 0.25rem (4px)
- sm: 0.5rem (8px)
- md: 1rem (16px)
- lg: 1.5rem (24px)
- xl: 2rem (32px)
- 2xl: 3rem (48px)

## 组件设计

### SearchAI 搜索框

- 输入框：2px 边框，聚焦时金色边框 + 阴影
- 搜索按钮：Primary 背景，悬停变亮
- 占位符：灰色文字

### LatestByCategory 分类导航

- 链接：Primary 颜色，悬停时金色下划线
- 分隔符：灰色竖线
- 响应式：移动端缩小字号和间距

### Explorer 文件树

- 树状线条：浅灰色连接线
- 展开动画：平滑过渡
- 悬停状态：背景色变化

### 卡片组件（Backlinks、RecentNotes 等）

- 默认：白色背景、细边框、sm 阴影
- 悬停：lg 阴影、上移 2px、金色边框
- 标签：圆角、灰色背景，accent 标签用金色

### 代码块

- 背景：#1e1e1e（深色）
- 圆角：8px
- 复制按钮：悬停显示，Primary 背景

### 表格

- 表头：Primary 颜色、灰色背景
- 行悬停：灰色背景
- 边框：底部细线

### 按钮

- 主要：Primary 背景、白色文字
- 强调：Accent 背景、白色文字
- 轮廓：透明背景、Primary 边框

## 动效设计

### 滚动淡入

使用 Intersection Observer 实现：
- 元素初始：opacity: 0, translateY: 20px
- 进入视口：opacity: 1, translateY: 0
- 过渡：0.6s ease-out

### 悬停效果

- 卡片：阴影变大、上移、边框变色
- 按钮：背景变亮、阴影出现
- 链接：下划线出现

### 页面过渡

利用 Quartz SPA 模式，添加页面切换淡入效果。

## 响应式设计

### 移动端适配（<768px）

- 标题字号缩小
- 间距减小
- 色板改为 2 列
- 导航换行

## 文件清单

| 文件 | 改动类型 | 说明 |
|------|---------|------|
| `quartz/styles/custom.scss` | 主要修改 | 所有样式覆盖 |
| `quartz.config.ts` | 配置修改 | 字体、主题配置 |

## 验证标准

1. **构建成功**：`npx quartz build --directory vault-content` 无报错
2. **本地预览**：效果符合预览文件
3. **移动端兼容**：响应式布局正常
4. **性能无损**：构建时间无明显增加
5. **功能完整**：所有组件正常工作

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 字体加载慢 | 首屏性能 | 使用 `display: swap`，CDN 缓存 |
| CSS 变量兼容性 | 旧浏览器 | 渐进增强，提供回退值 |
| 样式冲突 | 组件异常 | 使用 Quartz 的 `custom.scss` 机制 |

## 实施顺序

1. 定义 CSS 变量主题系统
2. 配置字体系统
3. 增强排版样式
4. 增强组件样式
5. 添加动效系统
6. 细节打磨
7. 验证测试
